## 📋 درباره پروژه

این پروژه یک **سیستم تشخیص اعداد** است که با استفاده از **شبکه‌های عصبی کانولوشنی (CNN)** پیاده‌سازی شده است. تشخیص اعداد یکی از چالش‌های کلاسیک در حوزه بینایی ماشین است. در این پروژه، ما یک مدل عمیق آموزش‌داده‌شده داریم که می‌تواند یک تصویر ورودی را پردازش کند، تک‌تک اعداد داخل آن را پیدا کرده، جدا کند و در نهایت هر عدد را به‌درستی تشخیص دهد. این قابلیت در پردازش فرم‌ها، شناسایی پلاک‌ها و اپلیکیشن‌ها کاربرد گسترده‌ای دارد.

### ✨ قابلیت‌های کلیدی

- **پیش‌پردازش هوشمند تصاویر:** شامل تبدیل به خاکستری، دودویی‌سازی (Binary)، نویزگیری و تغییر اندازه استاندارد با استفاده از OpenCV.
- **استخراج اعداد (Digit Extraction):** تشخیص کانتورها (Contours) در تصویر برای پیدا کردن محل دقیق هر عدد و برش (Crop) آن‌ها.
- **معماری قدرتمند CNN:** استفاده از دو لایه کانولوشن + ماکس‌پولینگ برای استخراج بهترین ویژگی‌ها از تصاویر اعداد.
- **آموزش و ارزیابی:** آموزش مدل بر روی مجموعه‌داده استاندارد و نمایش معیارهای عملکرد.
- **ذخیره و بازیابی مدل:** ذخیره مدل آموزش‌دیده در قالب فایل `H5` برای استفاده مجدد بدون نیاز به آموزش دوباره.

## 🛠️ تکنولوژی‌های استفاده‌شده و نقش آن‌ها

در این پروژه از کتابخانه‌های متنوعی استفاده شده است که هرکدام وظیفه خاصی را بر عهده دارند:

- **TensorFlow / Keras:** هسته اصلی پروژه برای ساخت، کامپایل و آموزش مدل شبکه عصبی CNN.
- **OpenCV (cv2):** سنگ‌بنای پردازش تصویر. برای خواندن تصاویر، تبدیل آن‌ها به فرمت باینری، تشخیص کانتورها و استخراج اعداد از تصویر اصلی استفاده می‌شود.
- **NumPy:** قلب محاسبات ریاضی در پایتون. در این پروژه از NumPy برای تبدیل تصاویر به آرایه‌های عددی (اندازه‌گذاری پیکسل‌ها)، تغییر شکل (Reshape) داده‌ها برای ورودی به شبکه و انجام عملیات‌های برداری روی ماتریس‌ها استفاده شده است.
- **Matplotlib:** کتابخانه مصورسازی داده. در مرحله آموزش برای رسم نمودارهای **دقت (Accuracy)** و **مقدار خطا (Loss)** در طول دوره‌های آموزشی استفاده می‌شود. همچنین در بخش استخراج، برای نمایش تصویر اصلی و اعداد برش‌خورده به‌صورت کنار هم، از Matplotlib بهره گرفته شده است.
- **scikit-learn:** برای تقسیم‌بندی داده‌ها به مجموعه‌های آموزش و تست (Train/Test Split) و همچنین نمایش ماتریس درهم‌ریختگی (Confusion Matrix) جهت ارزیابی عملکرد مدل.
- **Jupyter:** محیط تعاملی برای اجرای مرحله‌به‌مرحله کدها و مشاهده خروجی آنی.
